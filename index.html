Informe: Serie de Crecimiento en Octave

1. Introducción

Este informe presenta un programa en Octave que modela una serie de crecimiento, útil para representar fenómenos como inversión, población o usuarios.

2. Código

datos(1) = 100;

for i = 2:10
    datos(i) = datos(i-1) * 1.05;
end

disp(datos)
plot(datos)

3. Explicación

Inicialización

Se inicia con un valor de 100.

Ciclo

Se calcula cada valor usando el anterior.

Crecimiento

Cada valor aumenta un 5%, generando crecimiento exponencial.

Salida

Se muestran los datos y se grafican.

4. Ejecución en Octave

  1. Guardar como crecimiento.m
  2. Abrir Octave
  3. Ejecutar: crecimiento

5. Resultado esperado

100 105 110.25 115.76 ...

6. Conclusión

El programa muestra cómo modelar crecimiento en matemática aplicada.
